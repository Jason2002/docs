4、	Q&A:

Q：公积金联名卡具有哪些功能？
A：公积金联名卡具有两个功能：一为储蓄账户，二为公积金账户。公积金账户专门用于公积金缴交情况的查询（即：只能看不能取）；职工可以自己向储蓄账户里存入金额（和普通银行卡一样），同时经过公积金中心审批通过的公积金提取金额也是从公积金账户划转入储蓄账户的。

Q:是不是凭这张公积金联名卡，就可以随意领取公积金？
A:不是的，在联名卡中的职工公积金账户是冻结的，只有在公积金中心成功办理提取手续之后，这笔公积金支取款才会划转到领取人的联名卡储蓄账户上。
符合公积金提取条件的建议大家办理委托提取，这样公积金中心会在每年的固定时期将当年可提取额转到你的联名卡账户上。

Q:如何查询公积金账户的明细？
A:职工可以在公积金中心柜台、四川省住房公积金网站（http://www.scsjgjj.com/）、发卡银行ATM机和自助设备上，查询自己公积金缴存账户情况。也可拨打电话86742111用身份证号码进行自助查询。

Q:公积金联名卡是否有年费？费用是多少？
A:公积金联名卡是免收银行卡年费及小额账户管理费，需要注意的是，职工办理了个人住房公积金销户业务后，所持公积金联名卡将解约，不再享受银行卡年费减免优惠。
